My Favourite HTML Tags!

My top 5 list:

Button
The "button" tag is used to represent a clickable box.
Abbr
The "abbar" tag is used to abreviate text. When used, the user hovers over the abreviation and the text will appear.
!----
The "!----" tag is used when the web designer wants to create notes within the code and does not want the notes to be seen by the user.
Ol
The "ol" tag is used when creating ordered lists. It will place a number in front of each iteam title.
Nav
The "nav" tag is used to direct users using links within a document or to another documents.
