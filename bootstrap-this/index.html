About Bootstrap

Bootsrap is very powerful. Many web designers and web developpers use it. It is easy to work with becasue of its felxibilty and responsivabilty. The user will add a class attribute in the HTML to make the desiered changes to their site.

Colours

Text

Bootstrap gives the user the ability to easily change the text colour. To change the colour of the text the user must add 'text-...' in the class attribute. Some exaples that a user could use are:

  • text-primary
  • text-danger
  • text-info
  • text-white

Background

The background colour is easy to change similar to changing the text colour. To change the background colour the user needs to add 'bg-...' in the class attribute. Some exaples that a user could use are:

  • bg-primary
  • bg-danger
  • bg-info
  • bg-white

Spacing

How it works!

The user has the choice between margin and padding using Bootstrap. The user will use the format {property}{sides}-{size}.

The property is 'm' for margin and 'p' for padding.

The sides will be:

  • t - sets margin-top or padding-top
  • b - sets margin-bottom or padding-bottom
  • l - sets margin-left or padding-left
  • r - sets margin-right or padding-right
  • x - sets both left and right
  • y - both top and bottom
  • blank - sets margin or padding on all 4 sides

The size will be:

  • 0 - eliminates the margin or padding by setting it to 0
  • 1 - sets the margin or padding to $spacer * 0.25
  • 2 - sets the margin or padding to $spacer * 0.5
  • 3 - sets the margin or padding to $spacer * 1
  • 4 - sets the margin or padding to $spacer * 1.5
  • 5 - sets the margin or padding to $spacer * 3
  • auto - sets the margin to auto

Typography

Display Heading

The user can use this attribue to make their sites have more variaty. The atrribute will change fom 1 being the largest to 4 being the smallest of the headers. The user can chose on of these to add to their headers:

  • display-1
  • display-2
  • display-3
  • display-4

Other Typography

  • Users can add 'lead' to the class attribute to higlight text.
  • Users can use 'text-right', 'text-left', or 'text-center' to align content in the class attribute.
  • To add a b;ockquote a user will need to add 'blockquote' to the class attribute.
  • Users can add 'list-style' to a class attribute to change the apperance of the bullets in a list.
  • To add a source title a user can add 'blockquote-footer' to the class attribute.

Grid

How it works!

Grid will consist of containers, rows, and columns to layout and align content. Grid options will consist of different breakpoints. The breakpoints are:

  • Extra small - greater then 576px - 'col- '
  • Small - less or eaqual to 576px - 'col-sm- '
  • Medium - less or equal to 768px - 'col-lg-'
  • Extra large - less or equal to - 'col-xl- '

Combining Columns

The user can divide the row by 12 units at the end of the 'col' attribute. Some examples are:

ex ...class="col-md-6" ...class="col-md-6"

or ...class="col-sm-8" ...class="col-sm-4"

even ...class="col-lg-4" ...class="col-lg-4" ...class="col-lg-4"

Buttons

Colours

Like adding colours to text, the user will add 'btn-...' to the class attribute. The user could add 'btn-primary' for example to have a blue button. To add a coloured outline the user would add 'btn-outline-primary' to add a blue outline.

Button using ...class="btn-primary"

Button using ...class="btn-outline-primary"

Size

The user has the choice to change the button size. The user will add 'lg' to make the button larger or can add 'sm' to make the button smaller in the 'btn-' in the class attribute. An example would be:

large - ...class"btn-lg"

regular - ...class"btn"

small - ...class"btn-sm"
